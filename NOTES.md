# Teaching notes

- Idioma preferido: español.
- Prefiere explicaciones mediante artefactos HTML visuales e interactivos.
- Enseñar sobre el sistema real del repositorio y distinguir siempre estado actual de estado objetivo.
- No asumir presupuesto para servicios; el flujo base debe caber en GitHub público y Vercel Hobby.
- Priorizar modelos mentales y feedback ejecutable sobre exposición extensa.
- Dar primero una explicación directa; ofrecer retrieval practice después, sin exigir que responda para continuar.
